🤖 AIでまとめて入力する

① 下の「指示文をコピー」を押す

② ChatGPTやClaudeに貼りつけ、成績表のスクショを一緒に送る

③ 返ってきたJSONを「.json」ファイルとして保存する（AIに「ファイルにして」と頼んでもOK）

④ ホーム → ⚙️設定 → 📥取り込む でそのファイルを選ぶ

あなたは成績データ入力の係です。渡した浜学園の成績表（スクリーンショット）から「あなたの成績」を読み取り、次の形式のJSONだけを出力してください。説明文は不要です。

{
  "app": "chuju-mama-banso",
  "ver": 1,
  "seiseki": {
    "kokai": [
      {
        "id": "k2026-04",
        "ym": "2026-04",
        "class": "西宮V1",
        "koku": { "dev": 58.3, "score": 78, "rank": 210 },
        "san":  { "dev": 65.1, "score": 92, "rank": 45 },
        "ri":   { "dev": 60.2, "score": 80, "rank": 120 },
        "sha":  { "dev": 57.0, "score": 70, "rank": 300 },
        "avg4": { "dev": 61.0 }, "rank4": "40/1500",
        "avg3": { "dev": 62.0 }, "rank3": "27/2884",
        "avg2": { "dev": 61.5 }, "rank2": "131/2884",
        "kijutsu": { "dev": 55.0, "score": 40 }
      }
    ],
    "fukushu": [
      { "id": "f2026-06-01-san12", "date": "2026-06-01", "subject": "算数", "no": 12, "score": 85 }
    ],
    "saire": [
      { "id": "s2026-06-07", "date": "2026-06-07", "no": 5, "keisan": 90, "fukushu": 110 }
    ],
    "saikoku": [
      { "id": "k2026-06-07", "date": "2026-06-07", "no": 8, "score": 85 }
    ],
    "nadago": [
      { "id": "n2026-06-28", "date": "2026-06-28", "no": 2, "koku": 75, "san": 100, "ri": 94, "kokuRank": 5, "sanRank": 1, "riRank": 3 }
    ],
    "atotal": [
      { "id": "a2026-05-master", "ym": "2026-05", "course": "マスター", "rank": 1 }
    ]
  }
}

ルール:
- kokai＝公開学力テスト（月ごと1件）。ym は "YYYY-MM"。dev=偏差値、score=得点、rank=順位。rank4/rank3/rank2 は「順位/受験者数」の文字列。kijutsu＝国語記述。sha（社会）と avg4/rank4（4科）は成績表にあるときだけ入れる（無ければ省略）。
- fukushu＝復習テスト。subject は 算数/国語/理科 など。no=第何回、score=得点。
- saire＝最高レベル特訓算数。keisan=計算テストの得点、fukushu=復習テストの得点。
- saikoku＝最高レベル特訓国語。復習テストのみ（100点満点）。score=得点、no=第何回（実力テストは null）。
- nadago＝灘中合格特訓（灘合）。koku/san/ri は各教科の得点。kokuRank/sanRank/riRank は各教科のクラス内順位（成績表にあるときだけ）。
- atotal＝A表のクラス内順位。course は マスター/最レ算数/最レ国語/灘合 のどれか。rank=その月度のクラス内順位。ym は "YYYY-MM"（月度）。総合点は復テ・公開の記録から自動計算されるので入力不要（A表個人票の実物値 fuku/kokai/total を入れた場合はそちらを優先表示。灘合の総合点は実物値がある場合のみグラフに出ます）。
- 読み取れない値は null。数値は数値型で書き、単位（点・位）は付けない。
- id は他とかぶらない英数字の文字列。上の例のように日付から作る。
- 該当データのない配列は [] のままでよい。

同じ月の公開テストは上書き、その他は id が同じものが上書きされます。何度取り込んでも重複しません。
